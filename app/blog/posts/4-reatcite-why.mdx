##### 0614 update

搞懂了 [Vue Reactive](https://github.com/kaonlee/vue3-app/blob/main/src/views/ReactiveTest.vue)  由 Vue 觸發機制才會有完整的監聽。

實際的例子為：

設計了一個 GameLoop 的 Class

因為在 Class 中使用了 [requestAnimationFrame()](https://developer.mozilla.org/zh-TW/docs/Web/API/Window/requestAnimationFrame) ，由於傳入的用例會是

```js
requestAnimationFrame(this.tick) // this.tick is a method of Class
```

直接使用的話 this 會指向 requestAnimationFrame 的 this (window)。
故需要綁定 `this.tick = this.tick.bind(this)`

也因此會觸發無法及時 reactive 的問題。

之後應該寫一篇筆記來記錄這件事情。
